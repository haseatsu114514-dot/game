# Riders Rescue (Stage 1)

スマホ対応の高難易度2Dアクション（ブラウザゲーム）です。PS1風のドット画質に調整しています。

## 仕様
- 主人公: りら（黒髪ボブ＋前髪 / 茶色の目 / 色白 / ライダースジャケット）
- 目的: 彼氏に甘い話を持ちかける悪の組織から救出し、会場最奥の白髪・白ヒゲの神を倒す
- プロテイン: 取るほどパワーアップ（移動速度・ジャンプ・キックの爽快感が上昇）
- バイク: 触れると15秒無敵（レア配置 / りらがバイクに乗って虹色エフェクト）
- アイテム量: プロテイン40個 / バイク4台 / 武器11個
- 導入: 開始時に短いムービー（スキップ可）
- ボス前演出: ホームパーティー会場のマンション突入中に白ヒゲの神が降臨するムービー（スキップ可）
- ストーリー: りら（AI / ジムインストラクター）が自己紹介（シュークリームで元気回復 / バニラプロテイン派）の直後、彼氏がホームパーティー会場へ連行される
- 接触時: 敵に触れると被弾。キックボタンで攻撃して吹き飛ばす
- キック演出: ヒットストップ / 画面揺れ / 火花 / 画面フラッシュ / コンボ表示で爽快感を強化
- ボイス: キック時と被弾時に、機械っぽい音質低めのシンプルボイスを再生
- 攻撃: 波動拳は廃止。キックボタン主体アクション
- 通常敵追加: 道中に「突進してくる孔雀」タイプが出現
- トラップ調整: 飛び出しトゲは撤廃
- 武器アイテム(1): ハンマーを取ると10秒、攻撃ボタンで上下叩き連打が強化
- 武器アイテム(2): グローブを取ると10秒、攻撃ボタンで前方百裂拳が強化
- ライフ: 3ハート制（被弾で1ハート減少、0で死亡）
- 踏みつけ: 上から敵を踏むと撃破できる
- 画質: PS1風ドット（陰影強化スプライト + 都会ネオン背景 + 走査線エフェクト）
- 難易度: やや難しい（覚えれば避けられる罠中心 / 慣れれば15分前後でクリア目安）
- ステージ長: 1面構成のロングステージ（終盤でボス戦）
- 世界観: 都会の夜ステージ
- BGM: ステージBGMは `stage_bgm.mp3` を再生（無敵終了後は先頭から再開）
- エンディング: 最奥到達でボス出現。撃破後に「お姫様抱っこ → 結婚 → ED」

## 実行
1. このフォルダでローカルサーバーを起動
2. ブラウザで `index.html` を開く

例:
```bash
cd /Users/hasegawaatsuki/Documents/New\ project/riders_rescue
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## 操作
- スマホ: 画面下のタッチボタン（左右 / KICK / JUMP）
- PC:
  - 移動: `←` `→` または `A` `D`
  - ジャンプ: `Space`
  - キック: `J` または `F`
  - ムービースキップ: `Enter`
